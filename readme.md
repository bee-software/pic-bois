#requirements
- Install python 2.7
- Install virtualenv: sudo pip install virtualenv
- Install fabric: sudo pip install fabric
- Install node : http://nodejs.org
- Install phantomjs: npm install phantomjs

#setup
- fab setup

#to run lint & tests
- fab

# to start the server
- fab start
